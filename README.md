# test-task-idftechnology

## User List App
### Используемые технологии:
- Kotlin
- Jetpack Compose
- Ktor
- Hilt
- Coroutines + Flow
- MVVM - архитектурный паттерн

В проекте есть Unit-тесты для UserViewModel.
